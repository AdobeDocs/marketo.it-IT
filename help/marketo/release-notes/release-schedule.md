# Pianificazione rilascio {#release-schedule}

**Rilascio**: Marketo utilizza una pianificazione trimestrale delle versioni principali. Una volta disponibili, ciascuna nota sulla versione sarà collegata nella tabella seguente.

**Data** di rilascio: L&#39;implementazione inizia in genere il mercoledì della settimana di rilascio, con la maggior parte dell&#39;aggiornamento che avviene il venerdì sera. Viene inviato un messaggio e-mail con date e dettagli di rilascio specifici. La data indicata corrisponde all&#39;aggiornamento del venerdì.

**Avanzamento**:

* In programma - Prevediamo di rilasciare le funzioni e i miglioramenti trimestrali alla data indicata
* In corso - Stiamo sviluppando e documentando il contenuto ora per la pubblicazione alla data indicata
* Spedito - Il rilascio è completo

Le date di rilascio sono soggette a modifica.
