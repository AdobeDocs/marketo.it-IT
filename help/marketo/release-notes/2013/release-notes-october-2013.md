# Note sulla versione: Ottobre 2013 {#release-notes-october}

Le seguenti funzionalità sono incluse nella release di ottobre 2013. Controlla la disponibilità delle funzioni [Marketo Edition](http://docs.marketo.com/display/docs/assets/pricing.php) .

## templates.marketo.com {#templates-marketo-com}

[Templates.marketo.com](http://templates.marketo.com) mostra i modelli delle e-mail e delle pagine di destinazione (inclusi i modelli di e-mail per dispositivi mobili reattivi) che potete scaricare dalla libreria del programma Marketo. Aggiungeremo modelli mensili, check-back spesso!

## developers.marketo.com {#developers-marketo-com}

[Developers.marketo.com](http://developers.marketo.com) è per gli sviluppatori che desiderano creare integrazioni in Marketo. Potete fare riferimento a diverse opzioni di integrazione, tra cui API JavaScript Munchkin, esempi di codice API SOAP, webhooks e script e-mail. Un SDK Java è disponibile anche su [GitHub](https://github.com/Marketo/SOAP-API-Java-Client).

## Scheda evento BrightTALK aggiornata {#updated-brighttalk-event-adapter}

Sincronizzate altri campi da BrightTALK a Marketo, inclusi Nome società, Titolo processo, Industria e Dimensione società.

## App per il check-in degli eventi del tablet Android {#android-tablet-event-check-in-app}

Controllate gli utenti registrati nell&#39;evento utilizzando la nostra nuova app [di archiviazione basata su](../../product-docs/core-marketo-concepts/mobile-apps/event-check-in/check-people-into-your-event-from-your-tablet.md) Android disponibile su [Google Play](https://play.google.com/store/apps/details?id=com.marketo.eventcheckin&amp;hl=en).
