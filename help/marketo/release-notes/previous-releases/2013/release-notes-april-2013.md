# Note sulla versione: Aprile 2013 {#release-notes-april}

Le seguenti funzionalità sono incluse nella versione di aprile.

## Integrazione Box {#box-integration}

Collega Marketo con il tuo account Box per copiare facilmente i file nello studio di progettazione.

![](assets/image2014-9-22-15-3a47-3a56.png)

## Plug-in Gmail {#gmail-plugin}

Se utilizzi Marketo Sales Insight, così come Gmail, puoi installare il nostro nuovo plugin Gmail attraverso lo store Chrome. Il plug-in consente di registrare i messaggi con Marketo, caricare i modelli e-mail di Marketo e inviare messaggi con le funzioni di tracciamento di Marketo.

![](assets/image2014-9-22-15-3a48-3a57.png)

## Analisi e-mail {#email-analysis}

Crea report e-mail avanzati in Revenue Explorer, come il report Click Activity Heat Grid (Fai clic su griglia di calore attività). Questo rapporto fornisce informazioni sul giorno e l’ora in cui le persone fanno clic sui collegamenti nelle tue e-mail.

La funzione Analisi e-mail nel suo complesso verrà attivata in più fasi nei mesi di aprile e maggio, durante la migrazione dei dati e-mail 2012 e 2013. In altre parole, alcuni clienti avranno accesso a questa funzione prima di altri.

![](assets/image2014-9-22-15-3a49-3a16.png)

## API del programma {#program-apis}

Supporto per i programmi nella chiamata API SOAP, incluso l’accesso in sola lettura ai dati del programma come: conta dei membri del programma, acquisiti da, successo, impostazioni, canali, tag, token e costi. Per ulteriori informazioni, consulta la documentazione API SOAP .

## Miglioramento di ON24 {#on-enhancement}

Il titolo del lavoro e il nome della società verranno sincronizzati su ON24 dal modulo di registrazione Marketo.
