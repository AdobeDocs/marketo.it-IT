# Scaricare un HTML di e-mail {#download-an-emails-html}

Per backup e altri scopi, Marketo consente di scaricare il contenuto HTML di un’e-mail.

1. Trova l’e-mail e selezionala. In **Azioni e-mail** clic a discesa **Scarica HTML**.
