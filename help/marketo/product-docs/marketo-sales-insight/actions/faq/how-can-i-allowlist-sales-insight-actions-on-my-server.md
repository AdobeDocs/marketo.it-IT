# Inserire nell&#39;elenco Consentiti Come posso [!DNL Sales Insight Actions] sul mio server? {#how-can-i-allowlist-sales-insight-actions-on-my-server}

Purtroppo non è possibile fornire un indirizzo IP statico. [!DNL Sales Insight Actions] viene distribuito nel cloud e utilizza oltre 200 server per elaborare le e-mail e le richieste degli utenti. E fornire un range molto ampio non sarebbe più sicuro.
