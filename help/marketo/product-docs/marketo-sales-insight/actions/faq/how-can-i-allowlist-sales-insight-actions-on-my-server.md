# Come posso Inserire nell&#39;elenco Consentiti azioni di approfondimento sulle vendite sul mio server? {#how-can-i-allowlist-sales-insight-actions-on-my-server}

Purtroppo non è possibile fornire un indirizzo IP statico. Sales Insight Actions è implementato nel cloud e utilizza oltre 200 server per elaborare le e-mail e le richieste degli utenti. E fornire un range molto ampio non sarebbe più sicuro.
