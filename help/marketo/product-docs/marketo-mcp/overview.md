# Documento panoramica MCP {#overview}

TESTO DEL PADRE:

Il server MCP pubblico per Marketo Engage è un servizio accessibile al cloud che espone la piena potenza delle API REST pubbliche di Marketo Engage come strumenti strutturati e individuabili per gli agenti di intelligenza artificiale. Basato sul Model Context Protocol (MCP), trasforma le funzionalità di automazione marketing di Marketo in un’interfaccia standardizzata e basata su schema utilizzabile in modo sicuro e intelligente dai sistemi di intelligenza artificiale.

Invece di integrare manualmente gli endpoint Marketo in ogni applicazione, il server MCP agisce come un unico gateway AInativo, consentendo agli agenti compatibili con MCP di individuare le operazioni disponibili, comprendere i parametri richiesti ed eseguire azioni attraverso un protocollo coerente.

Sono disponibili una serie di strumenti che comprendono sia API REST pubbliche che API interne, e che eseguono le funzioni principali di Marketo come nell’interfaccia utente ma tramite chiamate API. Man mano che gli strumenti verranno sviluppati, verranno aggiunti ulteriori strumenti.

Il server MCP è offerto come self-service e può essere configurato tramite il framework Adobe IO creando un endpoint e autenticando la connessione. Per ulteriori informazioni e dettagli sulla configurazione, visita il sito Adobe Developer all’indirizzo (aggiungi il collegamento qui).

Gli strumenti di Campaign visualizzati come API di SOAP sono stati convertiti in API REST e sono stati aggiunti.
