# Documento panoramica MCP {#overview}

TESTO DEL PADRE:

Il server MCP pubblico di Marketo Engage è un servizio basato su cloud che rende le API REST di Marketo accessibili agli agenti di intelligenza artificiale come strumenti strutturati. Basato sul Model Context Protocol (MCP), fornisce un’interfaccia standardizzata basata su schema che consente ai sistemi di intelligenza artificiale di interagire con Marketo in modo sicuro ed efficiente.

Invece di integrare manualmente gli endpoint Marketo in ogni applicazione, il server MCP fornisce un unico gateway nativo per l’intelligenza artificiale, che consente agli agenti compatibili con MCP di individuare le operazioni disponibili, comprendere i parametri richiesti ed eseguire le azioni tramite un protocollo coerente.

Sono disponibili una serie di strumenti che comprendono sia API REST pubbliche che API interne, e che eseguono le funzioni principali di Marketo come nell’interfaccia utente ma tramite chiamate API. Man mano che gli strumenti verranno sviluppati, verranno aggiunti ulteriori strumenti. &lt;---- BISOGNO DI CHIARIMENTI

Il server MCP è offerto come self-service e può essere impostato tramite il framework I/O di Adobe creando un endpoint e autenticando la connessione. Per ulteriori informazioni e dettagli sulla configurazione, visita il sito Adobe Developer (aggiungi il collegamento qui). QUESTO? https://developer.adobe.com/marketo-apis/

Gli strumenti di Campaign visualizzati come API di SOAP sono stati convertiti in API REST e sono stati aggiunti.
