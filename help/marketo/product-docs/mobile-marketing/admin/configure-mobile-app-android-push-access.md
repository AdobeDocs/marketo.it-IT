# Configurare l’accesso push di Android per app mobili {#configure-mobile-app-android-push-access}

1. Fai clic su **[!UICONTROL Admin]**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. Seleziona **[!UICONTROL Mobile Apps]**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. Seleziona l’app mobile desiderata.

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. In **[!UICONTROL Push Access Type]**, seleziona **[!UICONTROL Android]** e fai clic su **[!UICONTROL Configure]**.

   ![](assets/image2016-6-15-15-3a16-3a22.png)

   >[!NOTE]
   >
   >Avrai bisogno di **[!UICONTROL Server API Key]** e **[!UICONTROL Project Number]** dallo sviluppatore di app mobili. Lo sviluppatore riceve questi messaggi accedendo a [!DNL Google Play Developer Console] per registrare l&#39;app e abilitare i messaggi cloud.

1. Immettere [!UICONTROL Server API Key] e [!UICONTROL Project Number]. Fai clic su **[!UICONTROL Save]**.

   ![](assets/image2015-4-22-18-3a54-3a54.png)

   Dolce. Assicurarsi di configurare l&#39;app con [!UICONTROL iOS].
