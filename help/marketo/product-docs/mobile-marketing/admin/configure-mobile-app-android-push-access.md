# Configurare l’accesso push di Android per app mobili {#configure-mobile-app-android-push-access}

1. Fai clic su **Amministratore**.

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. Seleziona **App mobili**.

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. Seleziona l’app mobile desiderata.

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. In **Tipo di accesso push**, seleziona **Android** e fai clic su **Configura.**

   ![](assets/image2016-6-15-15-3a16-3a22.png)

   >[!NOTE]
   >
   >Avrai bisogno di una **Chiave API server** e di un **Numero progetto** da parte dello sviluppatore di app mobili. Lo sviluppatore riceve queste notifiche accedendo a Google Play Developer Console per registrare l’app e abilitare i messaggi cloud.

1. Immetti la chiave API del server e il numero del progetto. Fai clic su **Salva**.

   ![](assets/image2015-4-22-18-3a54-3a54.png)

   Dolce. Assicurati di configurare l’app con iOS.
