# Inserire nell&#39;elenco Consentiti Come posso [!DNL Sales Connect] sul mio server? {#how-can-i-allowlist-sales-connect-on-my-server}

Purtroppo non è possibile fornire un indirizzo IP statico. [!DNL Sales Connect] viene distribuito nel cloud e utilizza oltre 200 server per elaborare le e-mail e le richieste degli utenti. E fornire una gamma molto ampia non sarebbe più sicuro per voi.
