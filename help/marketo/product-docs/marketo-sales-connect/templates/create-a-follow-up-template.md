# Creare un modello di follow-up {#create-a-follow-up-template}

Con la funzione Campagne, quando create un nuovo modello, aggiungete semplicemente il campo dinamico &quot;last_email_body&quot; nella parte inferiore del modello. In questo modo le e-mail verranno automaticamente filettate insieme nella campagna.
