# Panoramica di Web Personalization {#web-personalization-overview}

## Introduzione {#introduction}

Per i nuovi clienti Marketo, [!UICONTROL Real-Time Personalization] è diviso in quattro app, a cui si accede utilizzando le due sezioni seguenti: [!UICONTROL Web Personalization] e [!UICONTROL ContentAI].

![](assets/pasted-image-at-2016-03-23-02-45-pm.png) ![](assets/mlm-homepage-content-ai-281-29.png)

>[!NOTE]
>
>[!UICONTROL Web Personalization] e [!UICONTROL ContentAI] sono disponibili in tutte le lingue supportate da Marketo (inglese, francese, tedesco, giapponese, portoghese, spagnolo).

I clienti esistenti continuano ad accedere alle app tramite il riquadro Real-Time Personalization fino al rinnovo, momento in cui utilizzeranno l’accesso riportato sopra.

![](assets/image2016-2-9-8-3a52-3a32.png)

Quando si fa clic sul riquadro [!UICONTROL Web Personalization], viene visualizzata la pagina Dashboard. Fare clic sul riquadro [!UICONTROL ContentAI] per passare direttamente alla pagina Consigli sul contenuto.

Le quattro app sono:

* [!UICONTROL Web Personalization]
* [!UICONTROL Account-Based Web Marketing]
* [!UICONTROL Website Retargeting]
* [!UICONTROL ContentAI]

Il riquadro **[!UICONTROL Web Personalization]** è il tuo punto di accesso per la creazione di campagne web personalizzate. Marketing web basato su account e retargeting web. Puoi aggiungere contenuti alla pagina Content Analytics.

**[!UICONTROL Account-Based Web Marketing]** funziona con Web Personalization e dispone di Account denominati. La pagina Account denominati fornisce informazioni grafiche sui clienti esistenti, sugli impegni totali e sulle tendenze organizzative, nonché un elenco degli account. Puoi anche creare nuovi account denominati.

**[!UICONTROL Website Retargeting]** ti dà il potere di revendere a tipi di pubblico segmentati. Puoi impostare i segmenti, ma non utilizzare Account denominati nel filtro.

**[!UICONTROL ContentAI]** è il luogo in cui fornire consigli sui contenuti. Puoi aggiungere facilmente il contenuto alla pagina Content Recommendations (Consigli sui contenuti).

>[!NOTE]
>
>Per accedere a [!UICONTROL Account-Based Web Marketing] o [!UICONTROL Website Retargeting], è necessario fare clic sul riquadro [!UICONTROL Web Personalization], anche se non si dispone dell&#39;app [!UICONTROL Web Personalization].

## Organizzazioni {#organizations}

Nella scheda Organizzazioni vengono visualizzati tutti i dettagli (nome, posizione, attività e marca temporale) delle organizzazioni che hanno visitato il sito web durante un determinato periodo. La tabella può essere ordinata e organizzata in base all’ora, alla posizione, al dominio e tramite una ricerca a testo libero.

![](assets/image2014-11-10-19-3a23-3a18.png)

>[!TIP]
>
>&quot;Most Active&quot; (Più attivo) vs &quot;Latest&quot; (Più recente): la tabella è organizzata per organizzazioni identificate e quindi per provider di servizi Internet (identificati da un’icona ISP). Può essere ordinato in base a:
>
>* Più attive: le organizzazioni più attive nella tabella, in base al numero di visualizzazioni di pagina
>* Più recente: le organizzazioni più recenti nella tabella (impostazione predefinita)
>* Per informazioni preziose, filtra in base a più attivi

**Condividi organizzazioni**

Vuoi condividere le informazioni del visitatore del tuo sito web con un collega? Fai clic su un’organizzazione, quindi fai clic sul collegamento Condividi nel pannello a destra.

Attiva la sovrapposizione [!UICONTROL Share this Organization], in cui è possibile inviare un messaggio e-mail condividendo il nome e i dettagli dell&#39;organizzazione che ha visitato il sito Web.

![](assets/image2014-11-10-19-3a25-3a42.png)

## Visitatori {#visitors}

![](assets/wp-vis.jpg)
