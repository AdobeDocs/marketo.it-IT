# Attivare/disattivare sottoscrizioni di rapporti {#enable-disable-report-subscriptions}

La personalizzazione web dispone di diversi rapporti utili che vengono inviati tramite e-mail. Ecco come abbonarsi a loro.

1. Accedi alla personalizzazione web. Sotto il nome di accesso, fai clic su **Impostazioni utente**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. Seleziona il rapporto a cui vuoi abbonarti e la relativa frequenza. Fai clic su **Salva**.

   ![](assets/email-settings.png)

   Tutto qui! Per annullare l’iscrizione al rapporto, è sufficiente deselezionare e fare clic su **Salva**.
