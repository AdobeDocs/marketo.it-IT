# Abilita/Disabilita sottoscrizioni report {#enable-disable-report-subscriptions}

La personalizzazione web dispone di diversi rapporti utili che vengono inviati tramite e-mail. Ecco come abbonarsi a loro.

1. Accedi a Personalizzazione web. Sotto il nome di accesso, fai clic su **Impostazioni utente**.

   ![](assets/image2014-9-17-20-3a48-3a28.png)

1. Seleziona il rapporto a cui desideri abbonarti e la relativa frequenza. Clic **Salva**.

   ![](assets/email-settings.png)
