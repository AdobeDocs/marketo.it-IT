# Abilitazione o disabilitazione del tag JavaScript RTP {#enabling-or-disabling-the-rtp-javascript-tag}

Il tag JavaScript RTP controlla se la personalizzazione web tiene traccia dell’attività web oppure esegue campagne o contenuti consigliati sul sito web.

>[!NOTE]
>
>**Non è necessario rimuovere il tag dal codice html sul sito web.** Puoi controllarlo tramite le Impostazioni account.

## Attiva o disattiva il tag {#enable-or-disable-the-tag}

1. Vai a **Impostazioni account**.

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. In Configurazione dominio e dominio, in Tag, seleziona **Disattivato** per disattivare il tag JavaScript.

   ![](assets/account-settings-domain-tag.jpg)

   Quando il tag è impostato su Off, il codice JavaScript è disabilitato e Web Personalization non tiene traccia di alcuna attività web né esegue campagne o contenuti consigliati sul sito web.

1. Per abilitare il tag RTP, in Configurazione dominio e dominio, in Tag, seleziona il **On** attivare/disattivare.

   Facile.
