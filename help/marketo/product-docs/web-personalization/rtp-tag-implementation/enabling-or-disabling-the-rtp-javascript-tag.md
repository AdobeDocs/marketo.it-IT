# Abilitazione o disabilitazione del tag JavaScript RTP {#enabling-or-disabling-the-rtp-javascript-tag}

Il tag JavaScript RTP controlla se la personalizzazione web tiene traccia o meno dell’attività web o esegue campagne o consigli di contenuto sul sito web.

>[!NOTE]
>
>**Non è necessario rimuovere il tag dal codice HTML sul sito web.** Controllalo tramite le Impostazioni account.

## Attivare o disattivare il tag {#enable-or-disable-the-tag}

1. Vai a **Impostazioni account**.

   ![](assets/image2014-12-1-23-3a3-3a12.png)

1. In Configurazione dominio e dominio, in Tag selezionare **Disattivato** per disabilitare il tag JavaScript .

   ![](assets/account-settings-domain-tag.jpg)

   Quando il tag è impostato su Off, il codice Javascript è disabilitato e la personalizzazione web non tiene traccia di alcuna attività web né esegue campagne o consigli di contenuto sul tuo sito web.

1. Per abilitare il tag RTP, in Configurazione dominio e dominio, in Tag, seleziona la **On** alternare.

   Facile.
