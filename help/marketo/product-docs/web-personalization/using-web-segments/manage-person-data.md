# Gestire i dati delle persone {#manage-person-data}

Utilizza i dati personali per la personalizzazione web selezionando i campi persona da utilizzare nella segmentazione.

1. Vai a **Impostazioni account**.

   ![](assets/image2015-5-7-15-3a17-3a23.png)

1. Vai a **Database**.

   ![](assets/account-settings-dropdown-database.jpg)

## Aggiunta di un campo Nuova persona {#adding-a-new-person-field}

1. Seleziona **Campo da aggiungere** dall’elenco a discesa per aggiungere un campo dati persona all’elenco.

   ![](assets/add-a-person-field-hand.jpg)

   >[!NOTE]
   >
   >Un nuovo campo viene aggiunto in sospeso e può richiedere fino a 24 ore per l’attivazione.

## Eliminazione di un campo persona {#deleting-a-person-field}

1. Fai clic sull’icona Elimina ( ![—](assets/image2015-3-24-13-3a45-3a56.png)) per rimuovere un campo dall’elenco. Fai clic su **Sì** per confermare l’eliminazione del campo.

   ![](assets/web-engagement-settings-delete.jpg)

   >[!NOTE]
   >
   >**Gestione dei campi Dati personali**
   >
   >* È possibile includere solo i campi dati personali
   >* È possibile aggiungere fino a 30 campi dati personali
   >* L’aggiunta di nuovi campi può richiedere fino a 24 ore per l’attivazione
   >* La lunghezza massima dei tipi di stringa è di 255 caratteri
   >* I campi nascosti verranno rimossi automaticamente
