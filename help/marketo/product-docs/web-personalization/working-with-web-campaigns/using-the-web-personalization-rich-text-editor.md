# Utilizzo dell’editor Rich Text per la personalizzazione web {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

L’editor Rich Text per la personalizzazione web presente nella pagina Imposta campagne consente la formattazione del testo, il collegamento e l’inserimento di immagini. Include opzioni di formattazione simili a quelle dei processori di testo standard.

>I token web utilizzano [dati personali](/help/marketo/product-docs/web-personalization/using-web-segments/manage-person-data.md) dal database Marketo gestito in Personalizzazione Web in Impostazioni account > Database. Se il token web è un nuovo campo di database non ancora in Impostazioni account > Database, verrà aggiunto automaticamente e può richiedere fino a 24 ore per l&#39;attivazione.
