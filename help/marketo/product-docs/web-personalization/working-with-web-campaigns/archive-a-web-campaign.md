# Archiviare una campagna Web {#archive-a-web-campaign}

1. Vai a Campagne **Web**.

   ![](assets/one.jpg)

1. Fate clic sul menu a discesa dello stato della campagna Web desiderata e selezionate **Archivia**.

   ![](assets/two-3.png)

   >[!NOTE]
   >
