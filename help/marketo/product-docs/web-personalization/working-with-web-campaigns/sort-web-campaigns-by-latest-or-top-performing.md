# Ordina campagne web dalla più recente o da quella con prestazioni migliori {#sort-web-campaigns-by-latest-or-top-performing}

Puoi ordinare le campagne web in due modi diversi.

1. Passa a **[!UICONTROL Web Campaigns]**.

   ![](assets/web-campaigns-hand-1.jpg)

1. Seleziona un tipo di ordinamento.

   >[!NOTE]
   >
   >**Definizione**
   >
   >**[!UICONTROL Latest]**: verrà ordinato in base alla data di creazione della campagna. La campagna più recente in alto.
   >
   >**[!UICONTROL Top Performing]** - ordinerà le campagne in base alla frequenza di click-through. Massimo click-through in alto.

   ![](assets/image2016-11-4-13-3a34-3a59.png)

   Sì, è davvero così facile.
