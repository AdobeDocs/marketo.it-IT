# Avviare/sospendere una campagna web {#launch-pause-a-web-campaign}

Una campagna Web è una reazione personalizzata associata a un segmento specifico e può essere una [finestra di dialogo](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-dialog-web-campaign.md) sul sito Web, una [sostituzione zona](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/create-a-new-in-zone-web-campaign.md), una funzionalità widget o un avviso e-mail.

È possibile avviare o sospendere una campagna Web in due modi dalla pagina [!UICONTROL Web Campaigns] o dalla pagina [!UICONTROL Set Web Campaigns].

1. Fai clic su **[!UICONTROL Web Personalization]**.

   ![](assets/one-1.png)

1. Vai a **[!UICONTROL Web Campaigns]**.

   ![](assets/two-1.png)

1. Se la campagna è stata avviata, fare clic su **[!UICONTROL Launched]** per modificare lo stato della campagna in **[!UICONTROL Paused]**, per sospendere la campagna.

   ![](assets/image2014-11-26-17-3a26-3a38.png)

1. Se la campagna è in pausa, fare clic su **[!UICONTROL Paused]** per modificare lo stato della campagna in **[!UICONTROL Launched]**. Verrà attivata e avviata la campagna.

   ![](assets/image2014-11-26-17-3a28-3a59.png)

   >[!NOTE]
   >
   >Puoi anche modificare lo stato della campagna nella pagina Imposta campagna.

1. Nella campagna, fai clic su **[!UICONTROL Edit]**.

   ![](assets/image2014-11-26-17-3a31-3a37.png)

1. Fare clic su **[!UICONTROL Save]** per salvare e mettere in pausa la campagna. Fare clic su **[!UICONTROL Launch]** per avviare e attivare la campagna.

   ![](assets/image2014-11-26-17-3a32-3a48.png)
