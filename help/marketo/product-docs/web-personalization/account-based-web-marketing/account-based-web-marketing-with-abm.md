# Marketing Web basato su account con TAM {#account-based-web-marketing-with-TAM}

Marketo ha introdotto una nuova applicazione [Target Account Management](https://docs.marketo.com/display/docs/account+based+marketing) (TAM).

Quando si utilizza la personalizzazione Web di Marketo e la nuova gestione dell&#39;account di Target (TAM):

* TAM è l&#39;origine dei tuoi elenchi di account
* Gli elenchi di account vengono gestiti (creati, modificati ed eliminati) in TAM
* Gli elenchi degli account sono sincronizzati con la personalizzazione Web per l&#39;utilizzo nella segmentazione. Selezionateli dall&#39;elenco a discesa

   * **Nota**: La sincronizzazione può richiedere fino a 5 minuti

* In Web Personalization (Personalizzazione Web), puoi creare [segmenti di targeting account dall&#39;elenco account](/help/marketo/product-docs/web-personalization/account-based-web-marketing/create-a-new-account-list.md) creato in TAM e personalizzare il contenuto per questi account

## Per i clienti Web ABM esistenti {#for-existing-web-TAM-customers}

Se state già utilizzando Web ABM:

* È possibile creare elenchi di account in Web ABM _e_ in TAM
* È possibile creare, modificare ed eliminare gli elenchi di account in TAM _e_ in Web ABM
* Gli elenchi degli account (da Web ABM e TAM) vengono sincronizzati in Web Personalization (Personalizzazione Web) per l&#39;utilizzo in Segmentazione

   * **Nota**: La sincronizzazione può richiedere fino a 5 minuti

* In Web Personalization (Personalizzazione Web) potete creare segmenti per gli account da elenchi di account creati in _sia_ Web ABM che TAM e personalizzare il contenuto per questi account
* Potete caricare i file CSV nella pagina Elenco account personalizzazione Web
