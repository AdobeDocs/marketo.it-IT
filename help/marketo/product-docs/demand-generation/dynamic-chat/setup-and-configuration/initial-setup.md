# Configurazione iniziale {#initial-setup}

Segui i passaggi seguenti per configurare Dynamic Chat.

## Accedere ad Admin Console {#access-admin-console}

>[!NOTE]
>
>**Autorizzazioni amministratore richieste.**

1. Dopo l&#39;abilitazione di [!DNL Dynamic Chat] per l&#39;istanza di Marketo, l&#39;amministratore di sistema designato riceverà un&#39;e-mail di benvenuto. Nell&#39;e-mail, fare clic su **[!UICONTROL Get Started]**.

   ![](assets/initial-setup-1.png)

1. Se in precedenza hai effettuato l’accesso a un’applicazione con un Adobe ID, accedi direttamente a Adobe Admin Console. In caso contrario, [configura il tuo Adobe ID](https://helpx.adobe.com/it/manage-account/using/create-update-adobe-id.html){target="_blank"}.

   ![](assets/initial-setup-2.png)

## Aggiungi utenti {#add-users}

1. Dopo aver effettuato l’accesso a Admin Console, l’operazione successiva consiste nell’aggiungere utenti. Il processo [documentato](/help/marketo/product-docs/demand-generation/dynamic-chat/setup-and-configuration/add-or-remove-chat-users.md#add-a-chat-user){target="_blank"}.

È ora di [connettere Dynamic Chat a Marketo](/help/marketo/product-docs/demand-generation/dynamic-chat/integrations/adobe-marketo-engage.md){target="_blank"}.
