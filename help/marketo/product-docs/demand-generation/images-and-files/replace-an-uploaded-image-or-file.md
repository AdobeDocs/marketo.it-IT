# Sostituire un&#39;immagine o un file caricato {#replace-an-uploaded-image-or-file}

1. Vai a **Design Studio**.

   ![](assets/designstudio-6.png)

1. Fai clic su **Immagini e file**, quindi selezionare il file da sostituire.

   ![](assets/image2014-9-16-11-3a21-3a48.png)

   >[!TIP]
   >
   >È inoltre possibile [cercare immagini o file caricati](/help/marketo/product-docs/demand-generation/images-and-files/search-uploaded-images-and-files.md).

1. Fai clic su **Azioni file** e seleziona **Sostituisci immagine o file**.

   ![](assets/image2014-9-16-11-3a21-3a55.png)

1. Individuare il file del computer che sostituirà il file corrente, quindi selezionarlo.

   ![](assets/image2014-9-16-11-3a22-3a2.png)

   >[!NOTE]
   >
   >Se non è possibile selezionare un file da una cartella condivisa, salvare il file sul disco rigido e sceglierlo da lì.

1. Verifica che il file visualizzato accanto a **Scegli file** è corretto, quindi fai clic su **Sostituisci**.

   ![](assets/image2014-9-16-11-3a22-3a12.png)

>[!NOTE]
>
>Il nome del file di sostituzione verrà modificato per corrispondere al nome del file precedente.

Ce l&#39;hai fatta!
