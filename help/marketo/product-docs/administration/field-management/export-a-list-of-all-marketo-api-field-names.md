# Esporta un elenco di tutti i nomi di campo API di Marketo {#export-a-list-of-all-marketo-api-field-names}

Se si utilizza [!DNL SOAP API] o [!DNL Munchkin API], sarà necessario un elenco di tutti i campi e dei relativi nomi API.

>[!NOTE]
>
>**Autorizzazioni amministratore richieste**

1. Passare all&#39;area **[!UICONTROL Admin]**.

   ![](assets/export-a-list-of-all-marketo-api-field-names-1.png)

1. Fare clic su **[!UICONTROL Field Management]**.

   ![](assets/export-a-list-of-all-marketo-api-field-names-2.png)

1. Fare clic su **[!UICONTROL Export Field Names]** per scaricare il foglio di calcolo.

   ![](assets/export-a-list-of-all-marketo-api-field-names-3.png)

Ora disponi di un foglio di calcolo con un elenco di tutti i campi e dei relativi nomi API.

![](assets/export-a-list-of-all-marketo-api-field-names-4.png)

>[!NOTE]
>
>Il limite di caratteri per i nomi API MLM è 255.
