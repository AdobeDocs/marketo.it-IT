# Requisiti minimi di sistema {#minimum-system-requirements}

Se vi state chiedendo cosa avete bisogno di accedere e usare Marketo, ecco le informazioni che state cercando.

>[!NOTE]
>
>iOS e Android al momento non sono supportati.

* Computer o laptop con Windows, OS X o Linux
* Tastiera e mouse
* Monitor con una risoluzione minima di 1280 x 768
* Browser aggiornato, consultare [Browser supportati](/help/marketo/product-docs/administration/setup-administration/supported-browsers.md) per ulteriori informazioni
* Connessione Internet
