# Requisiti minimi di sistema {#minimum-system-requirements}

Se ti stai chiedendo a cosa devi accedere e usare Marketo, ecco le informazioni che stai cercando.

>[!NOTE]
>
>iOS e Android non ancora supportati.

* Computer o laptop con Windows, OS X o Linux
* Tastiera e mouse
* Monitor con una risoluzione minima di 1280 x 768
* Browser aggiornato. Per informazioni, consultate Browser [supportati](supported-browsers.md) .
* Connessione Internet
