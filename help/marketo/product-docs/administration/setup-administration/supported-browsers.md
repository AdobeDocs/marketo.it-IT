# Browser supportati {#supported-browsers}

Marketo supporta attualmente i seguenti browser:

| Browser | Versioni supportate |
|---|---|
| [Google Chrome](https://www.google.com/intl/en/chrome/browser/) | più recente |
| [Mozilla Firefox](https://www.mozilla.org/en-US/firefox/new/) | più recente |
| [Microsoft Edge](https://www.microsoft.com/en-us/windows/microsoft-edge) | più recente |

Per una migliore esperienza, **mantieni aggiornato il browser**!
