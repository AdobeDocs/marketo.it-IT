# Riprogrammare una campagna batch intelligente nella visualizzazione Pianificazione del programma {#reschedule-a-batch-smart-campaign-in-the-program-schedule-view}

Ripianificare l’esecuzione di una campagna intelligente batch? Trascinali e rilasciali nel futuro nella visualizzazione Pianificazione del programma.

1. Seleziona la campagna avanzata da ripianificare.

   ![](assets/image2015-5-19-12-3a8-3a28.png)

1. Trascina fino alla nuova data in cui desideri eseguire la campagna.

![](assets/image2015-5-19-12-3a12-3a1.png)

1. Fantastico! La campagna intelligente è stata ripianificata. L&#39;ora può anche essere modificata dalla **Data** nei dettagli della voce.

   ![](assets/image2015-5-19-12-3a15-3a38.png)
