# Creazione di sovrapposizioni personalizzate nella vista Pianificazione programma {#creating-custom-overlays-in-program-schedule-view}

Puoi creare sovrapposizioni personalizzate per visualizzare le voci pertinenti alle tue esigenze.

1. Fai clic sul menu a discesa **Agenda** .

   ![](assets/image2014-9-24-10-3a20-3a11.png)

1. Seleziona **Sovrapposizioni**.

   ![](assets/image2014-9-24-10-3a20-3a17.png)

1. Seleziona i tipi di voce che desideri rendere visibili nella sovrapposizione.

   ![](assets/image2014-9-24-10-3a20-3a26.png)

1. Puoi anche filtrare per [tag programma](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags/use-tags-in-a-program.md).

   ![](assets/image2014-9-24-10-3a20-3a32.png)
