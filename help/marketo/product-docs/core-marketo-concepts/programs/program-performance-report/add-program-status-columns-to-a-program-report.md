# Aggiunta di colonne di stato del programma a un report del programma {#add-program-status-columns-to-a-program-report}

Aggiungi le statistiche sullo stato del programma al tuo [rapporto prestazioni programma](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md).

1. Vai a AttivitÃ  di marketing (o Analytics).

   ![](assets/login-marketing-activities-2.png)

1. Seleziona il rapporto.

   ![](assets/emailperformance.jpg)

1. Fai clic su **Configurazione** e trascinate sul tag Canale.

   ![](assets/image2014-9-23-16-3a26-3a38.png)

1. Seleziona un canale in base al quale filtrare.

   ![](assets/image2014-9-23-16-3a26-3a48.png)

   >[!TIP]
   >
   >Per visualizzare le colonne di stato del programma, il report deve essere filtrato per _solo uno_ canale.

1. Selezionare l&#39;opzione per visualizzare le colonne dello stato del programma. Clic **Applica**.

   ![](assets/image2014-9-23-16-3a26-3a53.png)

1. Ce l&#39;hai fatta! Fare clic sulla scheda Report per visualizzare il report con le colonne di stato del programma.

   ![](assets/programreport.jpg)

>[!NOTE]
>
>Se non viene visualizzata una colonna per ogni stato nel programma, verificare di avere [ha selezionato le colonne da mostrare](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/select-report-columns.md) nella relazione.
