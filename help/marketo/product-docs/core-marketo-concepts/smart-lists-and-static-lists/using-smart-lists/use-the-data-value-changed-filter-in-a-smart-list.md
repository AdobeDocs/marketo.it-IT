# Utilizza il filtro del valore dati modificato in un elenco avanzato {#use-the-data-value-changed-filter-in-a-smart-list}

I valori cambiano costantemente e vengono aggiornati nel database Marketo. A volte invece di cercare un valore specifico, si desidera cercare le persone che hanno cambiato valore. Ecco come.

>[!PREREQUISITES]
>
>[Creare un elenco avanzato](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)

1. Vai all&#39;area **Attività di marketing**.

   ![](assets/ma.png)

1. Seleziona l’elenco avanzato e fai clic sulla scheda **Elenco avanzato** .

   ![](assets/two-1.png)

1. Trova il filtro **Valore dati modificato** e trascinalo sull&#39;area di lavoro.

   ![](assets/three-1.png)

1. Impostare **Attribute** sul campo in cui si desidera cercare le modifiche al valore dei dati.

   ![](assets/four.png)

Questo filtro cerca tutte le persone che hanno subito una modifica (o modifiche) al loro stato. Per ottenere una maggiore flessibilità, prova ad [aggiungere un vincolo a un filtro di elenco avanzato](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/using-smart-lists/add-a-constraint-to-a-smart-list-filter.md).
