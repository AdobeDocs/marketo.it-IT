# Utilizzare un Webhook in una campagna avanzata {#use-a-webhook-in-a-smart-campaign}

Per utilizzare un [gancio](https://developers.marketo.com/documentation/webhooks/), aggiungilo a un [campagna intelligente](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md) come azione di flusso.

>[!AVAILABILITY]
>
>Non tutti i clienti hanno acquistato questa funzionalità. Contatta il tuo rappresentante commerciale per i dettagli.

1. [Creare una campagna intelligente](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/creating-a-smart-campaign/create-a-new-smart-campaign.md).

   >[!NOTE]
   >
   >I webhook possono essere utilizzati solo nelle campagne di attivazione.

1. Vai a **Flusso** e trascina **Chiama Webhook** azione di flusso.

   ![](assets/image2014-9-22-15-3a8-3a2.png)

1. Selezionare il Webhook.

   ![](assets/image2014-9-22-15-3a8-3a5.png)

1. È inoltre possibile utilizzare i hook di un elenco smart.

   ![](assets/2017-05-02-10-54-38.png)

1. Infine, puoi utilizzare i hook di web in **Aggiungi scelta** in un passaggio di flusso.

   ![](assets/image2014-9-22-15-3a8-3a13.png)
