# Sincronizza persona con Microsoft {#sync-person-to-microsoft}

Questa funzione è disponibile solo per coloro che integrano Marketo con Microsoft Dynamics.

## Panoramica {#overview}

Questo passaggio del flusso inserirà le persone create da Marketo nel tuo Dynamics CRM.

![](assets/one.png)

## Utilizzo {#usage}

È possibile impostare un utente di Dynamics come proprietario della persona.

![](assets/two.png)

>[!NOTE]
>
>Quando utilizzi l’azione di flusso &quot;Sincronizza persona con Microsoft&quot; (solo in una campagna di trigger), il lead/contatto verrà creato in tempo reale in Dynamics.
